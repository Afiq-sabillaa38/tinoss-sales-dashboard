# Tinoss Technology Sales Dashboard

This project is a Laravel-based sales dashboard that visualizes sales data using Chart.js.

## Features

- Sales summary by Product Group
- Filter by Period
- Filter by Product
- Filter by Product Group
- Interactive charts
- Sales data table

## Technology Used

- Laravel
- MySQL
- Chart.js
- Bootstrap / CSS

## Installation

1. Clone the repository

git clone https://github.com/Afiq-sabillaa38/tinoss-sales-dashboard

2. Install dependencies

composer install

3. Setup environment

Copy `.env.example` to `.env`

4. Import database

Import the SQL file:

database/tinoss_dashboard.sql

Example:

mysql -u root -p tinoss_dashboard < database/tinoss_dashboard.sql

5. Run the project

php artisan serve

Then open:

http://127.0.0.1:8000
